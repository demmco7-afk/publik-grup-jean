BAGIAN 3 — Pasang Domain Custom Opsional – kalau mau pakai domain sendiri
9
Buka Settings → Domains di Vercel
Di dashboard Vercel, klik project kamu → tab Settings → menu Domains → klik Add dan masukkan domain kamu
vercel.com/usernamemu/publik-grup-jean/settings/domains
Add
grupjean.com
Invalid Configuration
Configure DNS
10
Atur DNS di Tempat Beli Domain
Login ke tempat kamu beli domain (Niagahoster, Domainesia, Namecheap, dll.) → DNS Management → tambahkan record berikut:
TYPE NAME / HOST VALUE / POINTS TO TTL
A @ 76.76.21.21 Auto
CNAME www cname.vercel-dns.com Auto
⏱️
Tunggu 5–30 menit untuk propagasi DNS. Setelah itu domain langsung aktif dengan SSL gratis dari Vercel!
11
Domain Aktif & SSL Otomatis ✅
Setelah DNS propagasi, domain kamu akan berubah status jadi Valid di Vercel. SSL (HTTPS) dipasang otomatis, gratis!
